# OpenSEO

OpenSEO is a multi-project SEO workspace for researching, monitoring, and
improving websites with shared data and agent-assisted workflows.

## Language

**Shared Organization**:
The single team-wide workspace that contains every Project and is visible to
every admitted teammate.
_Avoid_: Personal workspace, tenant

**Admin**:
A teammate permitted to change shared configuration, including Projects and
Integrations.
_Avoid_: Owner, superuser

**User**:
A teammate permitted to view every Project and run SEO workflows without
changing shared configuration.
_Avoid_: Member, viewer

**Project**:
A client SEO workspace that groups a website, target market, Integrations, and
analysis history.
_Avoid_: Account, tenant

**Integration**:
An external account or data source connected to the Shared Organization or one
of its Projects.
_Avoid_: Link, connector account
